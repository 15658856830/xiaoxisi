# 剪映PC版操作技能

## 时间线管理

### 轨道分层标准

| 轨道 | 用途 | 具体内容 |
|------|------|----------|
| V1 | 主画面 | 口播/接诊场景等主要视觉素材 |
| V2 | 辅助画面（B 路/画中画） | BGM 视频、病例截图、知识卡片等 |
| V3 | 花字/标题/标注 | 金句、重点词、数据标注 |
| V4 | 贴纸/装饰元素 | 箭头、高亮框、表情贴纸等 |
| A1 | 人声 | 对白/旁白，始终保持在最前层 |
| A2 | BGM | 背景音乐 |
| A3 | 音效 | 转场音效、重点提示音等 |

### 轨道操作规则
- 人声（A1）和 BGM（A2）之间留间隙，避免音量打架
- 关键信息处自动降低 BGM 音量（音频关键帧或音量包络线）
- V3 花字层保持与主画面时间轴对齐
- 增删轨道时保持 A1 > A2 > A3 的优先级不变

## 字幕处理

### 操作流程
1. 剪映「文本 → 智能字幕」识别语音 → 生成初版字幕
2. 逐段校对医学术语（手动修正识别错误）
3. 选中关键术语 → 「样式」中加粗/换色（推荐品牌色或橙色）
4. 统一字幕样式模板，保存为预设复用

### 注意事项
- 医学术语（如「适应证」「禁忌证」「糖皮质激素」）剪映可能识别错误，优先手动订正
- 每行字幕不超过 18 字（含标点），过长则手动断行
- 字幕底部安全边距保持在画面 10% 以内

## 音视频处理

### BGM 平衡
- 人声音轨音量 0dB（基准）
- BGM 音轨音量 -15dB ~ -20dB（背景级）
- 关键信息段落用「音量关键帧」将 BGM 降至 -25dB
- 视频开头 0.5 秒 BGM 淡入，结尾 1 秒淡出

### 封面图制作
1. 抖音：截取医生正面/半身画面 + 大字标题（36-48pt），高对比度
2. 小红书：知识卡片式封面（纯色底 + 标题 + 关键词摘要），干净排版
3. 使用 cover_maker.py 批量生成封面图（PIL/Pillow 驱动）
4. 封面图存储在 materials/images/ 下，按项目日期分类

## 排版设计规范

文字样式在全项目中保持一致：

- 字体：统一用剪映内置字体，避免字体缺失 — 标题推荐「优设标题黑」或「剪映黑体」，正文用「系统默认」
- 字号层级：主标题 36-48px，副标题 28-32px，正文 20-24px
- 强调方式：重点词改色（橙色/品牌色）或加粗，最多连续 5 字
- 花字样式：每期保持统一，不每期换风格
- 安全区：重要文字和画面元素保持在画面 90% 区域内，不被裁剪
