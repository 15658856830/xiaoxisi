# 短视频平台适配技能

## 抖音平台规范

### 格式要求
- 分辨率：1080x1920 (9:16)，竖屏全屏
- 时长：15-60 秒（最佳 30-45 秒）
- 帧率：30fps
- 编码：H.264，码率 10-15 Mbps
- 音频：AAC 320kbps

### 内容策略
- 常见形式：科普干货、门诊日常切片、患者真实反馈、行业误区纠正
- 节奏快，信息密度高，字幕大而醒目
- BGM 轻快专业，善用热门音效
- 前 3 秒必须有钩子：反常识观点、痛点提问、权威数据、视觉冲击
- 结构：钩子 + 主体（1-2 个知识点）+ 行动召唤（关注/转发）
- 每期只讲 1 个核心信息，不贪多
- 字幕大而醒目，字体 >= 28pt
- 封面：医生本人照片 + 大字标题，高对比

### 常用标签
#医学科普 #医生日常 #门诊 #XX科医生 #健康知识

## 小红书平台规范

### 格式要求
- 分辨率：1080x1344 (3:4) 或 1080x1920 (9:16)
- 时长：30-120 秒
- 帧率：25-30fps
- 编码：H.264，码率 10-15 Mbps
- 音频：AAC 320kbps

### 内容策略
- 常见形式：科普干货、门诊日常切片、患者真实反馈、行业误区纠正、知识卡片讲解、就诊流程展示、医生 Vlog
- 节奏快，信息密度高，字幕大而醒目
- BGM 轻快专业，善用热门音效
- 封面是生命线：决定点击率的关键
- 审美要求高：画面干净、色调统一、花字排版精致
- 文案密度比抖音高，正文可附详细文字说明

### 封面风格
- 知识卡片式封面（纯色底 + 标题 + 关键词）
- 文字排版精良，留白充分
- 封面图尺寸 3:4 (1080x1440)

### 常用标签
#医生 #医疗科普 #门诊日常 #健康知识 #XX科

## 医生宣传视频常见类型

| 类型 | 说明 | 适用平台 |
|------|------|----------|
| 科普干货 | 讲一个医学知识点，解决一个常见疑问 | 抖音为主 |
| 门诊日常 | 记录真实的接诊场景，拉近距离 | 抖音+小红书 |
| 患者反馈 | 患者自述就诊体验，建立信任 | 抖音+小红书 |
| 误区纠正 | 破除常见谣言，树立专业权威 | 抖音为主 |
| 就诊流程 | 展示挂号-看诊-检查-取药全流程 | 小红书为主 |
| 医生故事 | 为什么选择从医 / 印象深刻的患者 | 小红书为主 |
| 科室介绍 | 展示科室环境和设备，打消陌生感 | 双平台 |

## 双平台通用原则
- 医生形象：专业但不严肃，亲和但有边界感
- 内容合规：不承诺疗效、不对比同行、不涉及未批准的药品器械
- 人设统一：白大褂/诊室/科室场景为视觉锚点
- 每期一个核心信息点
