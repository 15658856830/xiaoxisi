# 导出质量检查技能

## 输出前逐项检查清单

每次导出前逐项核对，发现未达标立即停下调整：

### 内容完整性
- [ ] 前 3 秒有吸引钩子（抖音）/ 封面有足够吸引力（小红书）
- [ ] 视频内容完整无缺失片段
- [ ] 核心信息点已传达清楚，无遗漏

### 技术合规
- [ ] 画面比例正确（抖音 9:16，小红书 3:4 或 9:16）
- [ ] 帧率稳定（无忽快忽慢段）
- [ ] 分辨率无突变跳跃
- [ ] 画面无黑边/裁剪异常

### 字幕
- [ ] 无错别字（尤其医学术语）
- [ ] 医学术语校对完毕
- [ ] 重点词已加粗或换色
- [ ] 每行字幕 <= 18 字

### 音频
- [ ] 人声清晰可辨
- [ ] BGM 音量不压人声
- [ ] 关键信息处 BGM 已降低音量
- [ ] 开头无爆音/结尾无突然中断

### 封面
- [ ] 封面图已生成并对应平台风格
- [ ] 标题文字清晰可读

### 文件输出
- [ ] 文件名符合 日期_选题名_平台_v{N}.mp4 格式
- [ ] 输出文件存在且大小不为 0
- [ ] 输出路径正确（抖音版 -> output/抖音/, 小红书版 -> output/小红书/）
